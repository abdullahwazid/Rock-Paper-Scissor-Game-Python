🎮 Rock Paper Scissor Game (Python)

A simple Rock Paper Scissor game built using Python where you play against the computer. This project was created to practice Python basics such as loops, conditionals, user input, and the random module.

🚀 Features
Play against the computer
Random computer choice
Winner is decided automatically
Simple command-line interface

🛠️ Technologies Used
Python 3

▶️ How to Run
Prerequisites : 
Python 3.x installed on your system
Clone the Repository : 
git clone https://github.com/abdullahwazid/Rock-Paper-Scissor-Game-Python.git
Navigate to the Project Folder : 
cd Rock-Paper-Scissor-Game-Python
Run the Game : 
python main.py
